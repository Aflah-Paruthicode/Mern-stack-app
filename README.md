 # Product Management System

*This is my first project using **decoupled architecture**. the frontend and backend are completely seperate but work together through an api. also implemented a role based authentication flow.*

### What It Does
- **Public View:** anyone can see the list of products.
- **Admin Login:** a secure login system using jwt, that lets only admin to do Create,Update and Delete.

### Tech Stack  
- **Frontend:** React and Chakra UI,
- **Backend:** Node.js + Express.js
- **Database:** Mongodb.
- **Security:** Json Web Tokens,

### Deployment
- **Frontend:** is live on **Netlify**
- **Backend:** is live on **Render**

### Setup
1. Clone the project

   ```bash
   git clone https://github.com/Aflah-Paruthicode/Mern-stack-app.git
   ```
   
2. Navigate to frontend and backend folders

   ```bash
   cd ./frontend
   cd ./backend
   ```
   
3. Install dependencies
   
   ```bash
   npm install
   ```

4. Start building

   ```bash
   npm run dev
   ```
